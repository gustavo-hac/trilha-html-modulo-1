Introdução

Um documento HTML (HyperText Markup Language) é a base de qualquer página web. Ele define a estrutura e o conteúdo que os navegadores interpretam e exibem para os usuários.

Estrutura Básica de um Documento HTML

Todo arquivo HTML segue uma organização padronizada, composta por elementos essenciais que indicam o início, o conteúdo e o encerramento da página. Abaixo está o modelo básico:

<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
  </head>
  <body>
    <h1>Meu Primeiro HTML</h1>
    <p>Este é um exemplo de parágrafo.</p>
  </body>
</html>

Explicação dos Principais Elementos

  • <!DOCTYPE html>: Indica ao navegador que o documento segue o padrão HTML5.
  • <html>: Elemento raiz que contém todo o conteúdo da página.
  • <head>: Contém metadados, título e links para estilos e scripts.
  • <body>: Contém todo o conteúdo visível da página, como textos, imagens e botões.
