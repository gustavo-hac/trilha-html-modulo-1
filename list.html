Listas ordenadas e não ordenadas.

Introdução

No HTML, as listas são usadas para organizar informações de forma estruturada. Existem dois tipos principais de listas: ordenadas e não ordenadas. Cada uma possui uma função específica e um modo de exibição diferente nos navegadores.

Lista Ordenada (<ol>)

A lista ordenada apresenta os itens em uma sequência numerada. É usada quando a ordem dos elementos importa, como em instruções passo a passo ou classificações.

Exemplo:

<ol>
        <li>Ligar o computador</li>
        <li>Abrir o navegador</li>
        <li>Acessar o site desejado</li>
        </ol>

Resultado:

  1. Ligar o computador
  2. Abrir o navegador
  3. Acessar o site desejado

Lista Não Ordenada (<ul>)

A lista não ordenada exibe os itens com marcadores (geralmente pontos) e é usada quando a ordem dos elementos não é relevante, como em listas de compras, recursos ou tópicos gerais.

Exemplo:

<ul>
        <li>Teclado</li>
        <li>Mouse</li>
        <li>Monitor</li>
        </ul>

Resultado:

  • Teclado
  • Mouse
  • Monitor

Principais Diferenças

  • Ordenadas (<ol>): usam números ou letras para indicar sequência.
  • Não ordenadas (<ul>): usam símbolos como pontos ou círculos.
  • Uso: listas ordenadas servem para etapas ou hierarquias; listas não ordenadas servem para agrupamentos sem ordem específica.
